`create-react-app`에는 많은 사전설정들이 이미 세팅되어 있다.

예를 들어 create-react-app을 이용해서 작업을 할 때,

즉, create-react-app을 사용해서 어플리케이션을 만들면
개발 서버에 접근한다든가, 자동으로 새로고침을 시켜준다든가,
즉각적으로 어플리케이션 안에 CSS를 포함시켜 준다든가 하는 기능들이 있는 것.

그리고 우리가 웹사이트를 publish할 준비가 되면
create-react-app은 publish하는 명령어를 가지고 있기 때문에
코드를 압축하고, 좀 더 빠르게 만들어준다.

결과적으로 react 어플리케이션을 만들어야 할 때 create-react-app을 사용.

이 한 줄의 코드를 실행하기 위해 무엇을 해야하는지 살펴볼 것이다.

react application을 만들기 위해 커맨드 창 (또는 터미널) 에서 다음과 같이 입력한다.

> npx create-react-app (프로젝트 명)

디렉토리가 생성되면 프로젝트를 실행하기 위해 해당 디렉토리 내에서 다음과 같이 입력한다.

> npm run
> npm start
